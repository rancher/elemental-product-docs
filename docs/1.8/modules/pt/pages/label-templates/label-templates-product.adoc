= Variáveis do Modelo de Produto
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

As informações nestas variáveis referem-se a atributos do sistema.

Subconjunto dos dados de informações do sistema SMBIOS (tipo 1).

Para mais informações sobre SMBIOS, consulte a
https://www.dmtf.org/sites/default/files/standards/documents/DSP0134_3.7.1.pdf[SMSBIOS specification].
