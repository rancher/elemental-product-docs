= Variáveis descontinuadas dos Modelos de Rótulos
:revdate: 2025-09-08
:page-revdate: {revdate}

As famílias de variáveis *SMBIOS* e *Hardware* foram descontinuadas na versão 1.7.0 do Elemental Operator.

Embora essas famílias de variáveis ainda sejam renderizadas corretamente nas versões atuais do Elemental Operator, seu suporte será removido em futuras versões do Elemental Operator.

Se você ainda estiver usando essas antigas famílias de variáveis dos Modelos de Rótulos, por favor, mude para xref:label-templates/label-templates.adoc#_label_templates_variables[as mais novas disponíveis] se possível.

Para simplificar a adoção das novas variáveis, verifique as tabelas que mapeiam as variáveis descontinuadas para as novas suportadas:

* <<_hardware_labels_to_new_label_templates_variables_table,Rótulos de Hardware para novas variáveis dos Modelos de Rótulos>>
* <<_smbios_labels_to_new_label_templates_variable_table,Rótulos SMBIOS para novas variáveis dos Modelos de Rótulos>>
