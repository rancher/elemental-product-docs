= Visão geral da arquitetura
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 06-11-2025
:page-revdate: {revdate}

{elemental-product-name} é a combinação de dois projetos principais: o link:{elemental_toolkit_url}[{elemental_toolkit_name}] e o link:{elemental_operator_url}[{elemental_operator_name}].

O link:{elemental_toolkit_url}[{elemental_toolkit_name}] permite a instalação e atualizações do sistema operacional a partir de imagens de contêiner OCI, para que o sistema operacional possa ser distribuído e recuperado por meio de registros de contêineres.

O link:{elemental_operator_url}[{elemental_operator_name}] estende o Rancher com funcionalidades de provisionamento e gerenciamento de sistema operacional, aproveitando o {elemental-product-name} Toolkit.
Ele preenche a lacuna entre o link:{elemental_toolkit_url}[{elemental_toolkit_name}] e o https://documentation.suse.com/cloudnative/rancher-manager/latest/en/cluster-deployment/launch-kubernetes-with-rancher.html[provisionamento do Cluster Rancher RKE2/K3s], proporcionando uma experiência integrada ao passar de hosts sem sistema operacional para clusters Kubernetes totalmente configurados.

Isso é alcançado por meio de um xref:rancher-os-management/architecture/services/architecture-services.adoc[conjunto de serviços] oferecido por {elemental-product-name}, que se torna possível graças aos xref:rancher-os-management/architecture/architecture-components.adoc[componentes] emprestados do {elemental-product-name} Toolkit e dos projetos {elemental-product-name} Operator.
