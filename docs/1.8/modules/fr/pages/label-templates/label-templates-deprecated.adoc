= Variables dont la prise en charge a cessé dans les modèles d'étiquettes
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-09-08
:page-revdate: {revdate}

Les familles de variables *SMBIOS* et *Matériel* ont cessé d'être prises en charge dans l'Opérateur Élémentaire v1.7.0.

Bien que ces familles de variables soient encore correctement rendues dans les versions actuelles de l'Opérateur Élémentaire, leur prise en charge sera supprimée dans les futures versions de l'Opérateur Élémentaire.

Si vous utilisez encore ces anciennes familles de variables des modèles d'étiquettes, veuillez passer à xref:label-templates/label-templates.adoc#_label_templates_variables[les nouvelles disponibles] si possible.

Pour simplifier l'adoption des nouvelles variables, consultez les tableaux qui font correspondre les variables dont la prise en charge a cessé aux nouvelles prises en charge :

* <<_hardware_labels_to_new_label_templates_variables_table,Étiquettes Matériel aux nouvelles variables des modèles d'étiquettes>>
* <<_smbios_labels_to_new_label_templates_variable_table,Étiquettes SMBIOS aux nouvelles variables des modèles d'étiquettes>>
