= Variables de modèle de stockage
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables de modèle de stockage définissent les attributs des disques disponibles dans le système.

Les disques sont suivis à la fois par numéro (0, 1, 2, ...) et par nom de périphérique (_sda_, _vda_, _nvme0n1_, ...).

Par exemple, si `${Storage/Disks/2/Name}` est égal à `sda`, le modèle de disque peut être récupéré via les variables `${Storage/Disks/2/Model}` et `${Storage/Disks/sda/Model}`.

[NOTE]
====
Les disques suivis incluent également des lecteurs de disques optiques et des dispositifs amovibles (par exemple, des clés USB).
====

== Types de lecteurs de disques

|===
| Type | Description

| Disque dur
| Disque dur

| FDD
| Lecteur de disquettes

| ODD
| Lecteur de disque optique

| SSD
| Disque statique à semiconducteurs

| virtuel
| Lecteur virtuel, c'est-à-dire des dispositifs de boucle

| Inconnu
| type de lecteur inconnu
|===

== types de contrôleurs de stockage de disques

|===
| Type | Description

| IDE
| Électronique d'unité intégrée

| SCSI
| interface de système informatique de petite taille

| NVMe
| Mémoire non volatile Express

| MMC
| contrôleur multimédia (utilisé pour les dispositifs de stockage de téléphones mobiles)

| virtio
| contrôleur/driver de stockage virtualisé

| loop
| dispositif de boucle

| Inconnu
| type de contrôleur inconnu
|===
