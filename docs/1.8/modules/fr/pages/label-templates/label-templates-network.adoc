= Variables de modèle de réseau
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables de modèle de réseau définissent les attributs des interfaces réseau disponibles dans le système.

Les NIC sont suivies à la fois par numéro (0, 1, 2, ...) et par nom d'interface (_eth0_, _virbr0_, ...).

Les adresses IPv4 et IPv6 sont suivies par numéro (0, 1, ...) et les premières entrées d'adresses IPv4 et IPv6 (c'est-à-dire, IPv4Addresses.0 et IPv6Addresses.0) sont suivies dans les variables IPv4Address et IPv6Address.

Par exemple, si `${Network/NICs/2/Name}` est égal à `eth0`, alors l'adresse MAC de l'interface réseau peut être récupérée via les variables `${Network/NICs/2/MacAddress}` et `${Network/NICs/eth0/MacAddress}`.

La première adresse IPv4 (`eth0`) peut être récupérée avec les variables `${Network/NICs/eth0/IPv4Address}` et `${Network/NICs/eth0/IPv4Addresses/0}`. L'adresse IPv4 secondaire `eth0` peut être récupérée avec `${Network/NICs/eth0/IPv4Addresses/1}` à la place.
