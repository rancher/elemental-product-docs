= {elemental-product-name} 支持
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-03
:page-revdate: {revdate}

`elemental-support` 是一个实用程序，用于收集有关运行 {elemental-product-name} 堆栈的机器的所有信息。

在排查任何 {elemental-product-name} 节点的问题时，可以生成一个包含日志和配置文件的压缩文件。 +
请注意，*生成的 tar.gz 文件可能包含敏感信息*，例如管理集群的访问词元等。

== 要求

可选地，`elemental-support` 实用程序需要在目标机器上预装 `kubectl`，以便收集节点信息。这对于调试节点在 {elemental-product-name} 升级过程中出现的问题非常有用。 +
使用的 kubectl 配置将从 `KUBECONFIG` 环境变量中查找，或者在其缺失时，将使用 `/etc/rancher/k3s/k3s.yaml` 或 `/etc/rancher/rke2/rke2.yaml` 文件。

== 收集信息

一旦运行，`elemental-support` 将在同一目录中创建压缩文件。
