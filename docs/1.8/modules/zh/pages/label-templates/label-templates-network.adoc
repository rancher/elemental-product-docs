= 网络模板变量
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

在网络模板变量中收集的信息定义了系统中可用网络接口的属性。

网络接口卡（NIC）通过编号（0, 1, 2, ...）和接口名称（_eth0_, _virbr0_, ...）进行跟踪。

IPv4地址和IPv6地址通过编号（0, 1, ...）进行跟踪，首个IPv4和IPv6地址条目（即IPv4Addresses.0和IPv6Addresses.0）在IPv4Address和IPv6Address变量中进行跟踪。

例如，如果`${Network/NICs/2/Name}`等于`eth0`，则网络接口的MAC 地址可以通过`${Network/NICs/2/MacAddress}`和`${Network/NICs/eth0/MacAddress}`变量同时检索。

`eth0`（第一个）IPv4地址可以通过`${Network/NICs/eth0/IPv4Address}`和`${Network/NICs/eth0/IPv4Addresses/0}`变量同时检索。次要的`eth0` IPv4地址可以通过`${Network/NICs/eth0/IPv4Addresses/1}`进行检索。
