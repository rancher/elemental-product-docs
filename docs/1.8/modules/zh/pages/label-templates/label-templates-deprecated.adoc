= 标签模板的已弃用变量
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-09-08
:page-revdate: {revdate}

*SMBIOS* 和 *硬件* 变量系列在 Elemental Operator v1.7.0 中已被弃用。

虽然这些变量系列在当前的 Elemental Operator 版本中仍然正确呈现，但它们的支持将在未来的 Elemental Operator 版本中被去除。

如果您仍在使用这些旧的标签模板变量系列，请尽可能切换到 xref:label-templates/label-templates.adoc#_label_templates_variables[更新的可用变量系列]。

为了简化新变量的采用，请查看将已弃用变量映射到新支持变量的表格：

* <<_hardware_labels_to_new_label_templates_variables_table,硬件标签到新标签模板变量>>
* <<_smbios_labels_to_new_label_templates_variable_table,SMBIOS 标签到新标签模板变量>>
