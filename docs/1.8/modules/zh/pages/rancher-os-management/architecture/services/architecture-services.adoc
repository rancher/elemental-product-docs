= 服务
:revdate: 2025-09-08
:page-revdate: {revdate}

{elemental-product-name} 功能通过以下服务列表提供：

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[机器接入]：允许主机预装操作系统，并在 {elemental-product-name} 操作员目录中注册。
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetes 集群部署]：触发在 {elemental-product-name} 操作员目录中已注册的可用主机上创建 Kubernetes 集群。
