= Referencia de MachineInventorySelector
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2024-09-24
:page-revdate: {revdate}

Un MachineInventorySelector selecciona MachineInventories basándose en selectores aplicados (usualmente mediante la coincidencia de un patrón en los valores de las etiquetas de MachineInventory).

Los MachineInventorySelectors tienen dos condiciones:

* `InventoryReady`, se convierte en verdadero si el MachineInventorySelector ha encontrado un MachineInventory coincidente y se ha establecido con éxito como el propietario del MachineInventory.
* `Ready`, rastrea si el selector ya ha adoptado una máquina y ha iniciado el proceso de aprovisionamiento de kubernetes (iniciar nodo).
