= Variables de plantilla de red
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

La información recopilada en las variables de plantilla de red define los atributos de las interfaces de red disponibles en el sistema.

Las NICs se rastrean tanto por número (0, 1, 2, ...) como por nombre de interfaz (_eth0_, _virbr0_, ...).

Las direcciones IPv4 y IPv6 se rastrean por número (0, 1, ...) y las primeras entradas de dirección IPv4 e IPv6 (es decir, IPv4Addresses.0 e IPv6Addresses.0) se rastrean en las variables IPv4Address e IPv6Address.

Por ejemplo, si `${Network/NICs/2/Name}` es igual a `eth0`, entonces la dirección MAC de la interfaz de red se puede recuperar a través de las variables `${Network/NICs/2/MacAddress}` y `${Network/NICs/eth0/MacAddress}`.

La primera dirección IPv4 (`eth0`) se puede recuperar con las variables `${Network/NICs/eth0/IPv4Address}` y `${Network/NICs/eth0/IPv4Addresses/0}`. La dirección IPv4 secundaria `eth0` se puede recuperar con `${Network/NICs/eth0/IPv4Addresses/1}` en su lugar.
