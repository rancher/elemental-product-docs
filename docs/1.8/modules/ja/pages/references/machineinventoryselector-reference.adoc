= MachineInventorySelectorの参照
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2024-09-24
:page-revdate: {revdate}

MachineInventorySelectorは、適用されたセレクター（通常はMachineInventoryラベル値のパターンマッチング）に基づいてMachineInventoriesを選択します。

MachineInventorySelectors には2つの条件があります：

* `InventoryReady`は、MachineInventorySelectorが一致する MachineInventory を見つけ、自身をその MachineInventory の所有者として正しく設定した場合、trueとなります。
* `Ready`は、セレクターがすでにマシンを採用し、kubernetesプロビジョニングプロセス（ノード起動）を開始したかどうかを追跡します。
