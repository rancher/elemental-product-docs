= ネットワークテンプレート変数
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-15
:page-revdate: {revdate}

ネットワークテンプレート変数に収集された情報は、システム内で利用可能なネットワークインタフェースの属性を定義します。

NICは、番号（0、1、2、...）およびインターフェース名（_eth0_、_virbr0_、...）の両方で追跡されます。

IPv4アドレスとIPv6アドレスは、番号（0、1、...）で追跡され、最初のIPv4およびIPv6アドレスエントリ（すなわち、IPv4Addresses.0およびIPv6Addresses.0）は、IPv4AddressおよびIPv6Address変数で追跡されます。

例えば、`${Network/NICs/2/Name}`が`eth0`に等しい場合、ネットワークインタフェースのMACアドレスは、`${Network/NICs/2/MacAddress}`および`${Network/NICs/eth0/MacAddress}`の両方の変数を介して取得できます。

`eth0`（最初の）IPv4アドレスは、`${Network/NICs/eth0/IPv4Address}`および`${Network/NICs/eth0/IPv4Addresses/0}`の両方の変数で取得できます。二次の`eth0` IPv4アドレスは、代わりに`${Network/NICs/eth0/IPv4Addresses/1}`で取得できます。
