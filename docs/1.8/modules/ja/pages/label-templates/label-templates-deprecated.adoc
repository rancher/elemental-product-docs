= ラベルテンプレートの廃止された変数
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-09-08
:page-revdate: {revdate}

*SMBIOS* および *ハードウェア* 変数ファミリーは、Elemental Operator v1.7.0 で廃止されました。

これらの変数ファミリーは現在の Elemental Operator バージョンでは正しく表示されますが、将来の Elemental Operator リリースではサポートが削除される予定です。

古いラベルテンプレートの変数ファミリーをまだ使用している場合は、可能であれば xref:label-templates/label-templates.adoc#_label_templates_variables[新しい利用可能なもの] に切り替えてください。

新しい変数の採用を簡素化するために、廃止された変数と新しいサポートされている変数のマッピングを示す表を確認してください：

* <<_hardware_labels_to_new_label_templates_variables_table,ハードウェアラベルから新しいラベルテンプレートの変数へのマッピング>>
* <<_smbios_labels_to_new_label_templates_variable_table,SMBIOSラベルから新しいラベルテンプレートの変数へのマッピング>>
