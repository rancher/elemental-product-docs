{elemental-product-name} 機能は、以下のサービスのリストを通じて提供されます：

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[マシンオンボーディング]: ホストがOSのプロビジョニングを受け、{elemental-product-name} オペレーターカタログに登録されることを可能にします。
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetesクラスターのデプロイメント]: {elemental-product-name} オペレーターカタログに登録された利用可能なホスト上でKubernetesクラスターの作成を開始します。
