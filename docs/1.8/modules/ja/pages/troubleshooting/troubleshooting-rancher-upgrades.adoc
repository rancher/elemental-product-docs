= Rancherのアップグレードのトラブルシューティング
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
Rancher v2.7.2へのアップグレードは、{elemental-product-name}クラスターが定義されている場合に失敗します。Rancherポッドがクラッシュループにハマります（https://github.com/rancher/rancher/issues/41145).を参照してください）。
====

この問題は、少なくとも1つの{elemental-product-name}クラスターが定義されている場合にのみ発生することに注意してください。

この問題を回避するには、`dynamicschemas.management.cattle.io`という名前の空のリソース`machineinventoryselectortemplate`を作成してください。

[,shell]
----
kubectl apply -f - <<EOF
kind: DynamicSchema
apiVersion: management.cattle.io/v3
metadata:
  name: machineinventoryselectortemplate
EOF

----

クラッシュループが停止し、Rancherのアップグレードが正常に完了します。
