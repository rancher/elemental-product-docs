= Speicher-Template-Variablen
:revdate: 2025-08-15
:page-revdate: {revdate}

Die in den Speicher-Template-Variablen gesammelten Informationen definieren die Attribute der im System verfügbaren Festplatten.

Die Festplatten werden sowohl nach Nummer (0, 1, 2, ...) als auch nach Gerätename (_sda_, _vda_, _nvme0n1_, ...) verfolgt.

Wenn `${Storage/Disks/2/Name}` beispielsweise gleich `sda` ist, kann das Festplattenmodell sowohl über die `${Storage/Disks/2/Model}` als auch über die `${Storage/Disks/sda/Model}` Variablen abgerufen werden.

[NOTE]
====
Die verfolgten Festplatten umfassen auch optische Laufwerke und wechselbare Geräte (z. B. USB-Sticks).
====

== Festplattentypen

|===
| Typ | Beschreibung

| HDD
| Festplattenlaufwerk

| FDD
| Diskettenlaufwerk

| ODD
| Optisches Laufwerk

| SSD
| Solid State Drive

| virtual
| virtuelles Laufwerk, d. h. Loop-Geräte

| Unbekannt
| unbekannter Laufwerkstyp
|===

== Speicherkontroller-Typen von Festplatten

|===
| Typ | Beschreibung

| IDE
| Integrierte Laufwerkselektronik

| SCSI
| Small Computer System Interface

| NVMe
| Nichtflüchtiger Speicher-Express

| MMC
| Multimedia-Controller (verwendet für Speichergeräte von Mobiltelefonen)

| virtio
| Virtualisierter Speicherkontroller/-treiber

| loop
| Loop-Gerät

| Unbekannt
| unbekannter Kontroller-Typ
|===
